  [1XAbstract[101X
  [33X[0;0YThis  document  describes  the  package  [5XIMG[105X,  which  implements  in [5XGAP[105X the
  iterated monodromy groups of Nekrashevych. It depends on the package [5XFR[105X.[133X
  
  [33X[0;0YFor  comments or questions on [5XIMG[105X please contact the author; this package is
  still under development.[133X
